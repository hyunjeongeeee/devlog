  • SpringBoot
  • Javascript

1. 프로젝트 소개

스프링부트를 기반으로 개발된 강아지용품 쇼핑몰로, REST API를 통해 클라이언트와 서버 간의

효율적인 통신 구현 및 모놀리식 아키텍처를 통한 기능 통합 관리 용이성을 제공하는 시스템

2. 개발기간

23.11. ~ 23.12

3. 배포환경

AWS EC2 인스턴스 설정: 웹 서버를 구성하여 애플리케이션 호스팅

RDS 구성: MySQL DB를 구축하여 데이터 관리 및 저장

4. DBMS 리팩토링 ( Oracle → MySQL )

리팩토링 이유 ❓

⇒ 비용 절감: 오픈 소스 제공 (라이선스 비용 없음)

⇒ 적은 리소스 요구: 상대적으로 적은 리소스 요구,

작은 규모의 프로젝트나 개발 환경에서 가벼운 데이터베이스 솔루션으로 MySQL 선택 가능

5. 아이템 페이지 세션 관련 오류 해결

로그인하지 않은 상태에서 상품상세 페이지 접속 오류 발생

⇒ 원인: 아이템 페이지의 form 액션 태그에 Session에 아이디가 있을 경우만 작동하게 설계

⇒ 해결: 상품 정보 전달을 하는 form 액션 태그에 Session에 아이디가 있는지 여부를 확인하는

조건문(th:if) 추가하여 로그아웃 상태에서도 접근 가능하게 수정

6. 상품 등록 기능 구현

서버 측에서 해당 요청 수신 ⇒ 받은 데이터 디코딩 ⇒ 적절한 데이터베이스 쿼리 사용하여 상품 정보 저장

유연성과 확장성 고려 ⇒ RESTful API 사용하여 클라이언트와 서버 독립적 개발 가능
